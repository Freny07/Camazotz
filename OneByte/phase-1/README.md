# Design Anti-Patterns Used

- Multiple CTAs  
- Poor visual hierarchy  
- Hidden actions  
- Confusing navigation  
- Cognitive overload  
- Inconsistent components  

Explain which ones you used and where:
We have used,
- Poor visual hierarchy in question, home and profile page 
- Hidden actions by switching button and option labels in all pages
- Confusing navigation in question pages
- Cognitive overload in profile, question and home page
- Inconsistent components in question,home and login pages by hardcoding colors and themes
- In CTA, our Home page has very bad CTA design.

-changing images
